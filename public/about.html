How It Works

The interpolation tool works using simple linear interpolation. The formula for the computation is:

x=x1+((y-y1)*((x2-x1)/(y2-y1)))

This equation determines the value of x by using the slope between two known points. In this tool, the known points are called Point 1 and Point 2, while the unknown point is simply referred to as the Unknown. The equation determines the value of one coordinate of the unknown point by placing it on the line created by the two known points at the location of the target x or y value.

When the unknown point is between the two known points, finding its unknown value is called interpolation. If the unknown point is located beyond the two known points, this is called extrapolation, and follows the same procedure.

This tool also supports quadratic interpolation using the following equation:

y=(y1*(((x-x2)*(x-x3))/((x1-x2)*(x1-x3))))+ (y2*(((x-x1)*(x-x3))/((x2-x1)*(x2-x3))))+ (y3*(((x-x1)*(x-x2))/((x3-x1)*(x3-x2))))

It should be noted that because this equation only determines the location of a single point, the equation of the true quadratic curve is difficult to represent on the chart shown.
